IoT 노브로 블라인드/커튼 제어

노브를 돌려서 블라인드나 커튼의 열림 정도를 조절합니다.

선택하기

SmartThings Advanced Web App에서 기기 ID를 가져오세요.

Devices 페이지 열기
Advanced Web App → Devices → 해당 기기 클릭 → Device ID를 복사하세요.
Advanced Web App → Devices → 전구 클릭 → Device ID를 복사하세요.
주의사항

  • 반드시 SmartThings Advanced Web App에서 복사하세요.

  • 전구가 Switch Level 및 Color Temperature 기능을 지원하는지 확인하세요.

필수 정보를 모두 입력해야 다음으로 진행할 수 있습니다.

색온도 범위

K

※ 입력하신 범위에 맞춰 20단계 제어 수치가 자동 계산됩니다. 전구의 기술 사양에 명시된 색온도 범위를 정확히 입력해야 루틴의 오작동을 막을 수 있습니다.

색온도는 1,000K 이상 20,000K 이하로 입력하고, 최소값은 최대값보다 작아야 합니다.
